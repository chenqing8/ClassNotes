Ctrl+Shift+z	 展开所有子代码块

**vscode代码块js、vue、less**
```
    vue  ------vue模板
    cl   ------console.log()
    for  ------for循环
    fi   ------for in循环
    fo   ------for of循环
    dm   ------data.map循环
    /*m  ------方法注释
    /*f  ------文件头注释
    crg  ------js代码注释可折叠
    httpg------axios get请求
    httpp------axios post请求
    emit ------$emit()

    bo   ------bus.$on()
    utf  ------less 文件头
```
